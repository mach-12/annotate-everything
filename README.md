- Zustand (state management)
- Docker

# Installation

Create Virtual Environment and install dependencies
```bash 
python3.10 -m venv env
```

Activate Virtual Environment
```bash 
. env/bin/activate
```

Install Dependencies
```bash 
pip install -r backend/requirements.txt
```
